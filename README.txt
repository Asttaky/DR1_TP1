README - Análise das Músicas Mais Reproduzidas no Spotify em 2024
=================================================================

Este projeto é uma aplicação desenvolvida em Streamlit que analisa as músicas mais reproduzidas no Spotify em 2024. A aplicação utiliza um dataset que contém informações detalhadas sobre as músicas, artistas e número de reproduções no Spotify.

Estrutura da Aplicação
----------------------

A aplicação é composta pelos seguintes elementos:

1. **Título**: Introduz a aplicação e seu propósito.
2. **Cabeçalho**: Explora os dados de streaming.
3. **Subcabeçalho**: Fornece uma visão geral da aplicação.
4. **Texto Formatado**: Explica o contexto e o objetivo da análise.
5. **DataFrame**: Exibe os dados do dataset "Most Streamed Spotify Songs 2024".
6. **Tabela**: Apresenta as primeiras cinco músicas do dataset.
7. **Métricas**: Mostra informações chave sobre três músicas selecionadas.
8. **Comandos Magic**: Apresenta um título dinâmico e uma lista de músicas.
9. **Elementos Multimídia**: Inclui imagem, vídeo e áudio relacionados às músicas analisadas.
10. **Mensagem Final**: Celebra as funcionalidades do Streamlit com um markdown e um GIF animado.

Requisitos
----------

- Python 3.7 ou superior
- Streamlit
- Pandas
- Arquivos de mídia (imagem, vídeo e áudio)
- Dataset "Most Streamed Spotify Songs 2024.csv"

Instruções de Uso
-----------------

1. **Instalação das Dependências**: Certifique-se de ter o Python instalado. Em seguida, instale as bibliotecas necessárias executando:
pip install streamlit pandas


2. **Execução da Aplicação**: Para iniciar a aplicação, execute o comando:
Substitua `nome_do_seu_arquivo.py` pelo nome do arquivo Python onde o código da aplicação está armazenado.

3. **Visualização dos Dados**: A aplicação será aberta em seu navegador, onde você poderá explorar os dados, visualizar as métricas, e interagir com os elementos multimídia.

Arquivos Necessários
--------------------

- `Most Streamed Spotify Songs 2024.csv`: Dataset contendo as músicas mais reproduzidas no Spotify em 2024.
- `image.png`: Imagem ilustrativa de uma capa de álbum.
- `Kendrick Lamar - Not Like Us.mp4`: Vídeo relacionado a uma das músicas analisadas.
- `Kendrick Lamar - Not Like Us.mp3`: Áudio da música "Not Like Us" de Kendrick Lamar.
- `XOsX.gif`: GIF animado utilizado no encerramento da aplicação.
